# 교뿡문고

[demo](https://pvvng.github.io/hostBook/)

### 카카오 책 검색 API를 활용해서 만든 간단한 책 검색 웹서비스입니다.

### React-Query의 활용과 이해를 위해 시작한 프로젝트입니다.

- - -

### 1. 책 이미지 호버시 애니메이션이 적용됩니다. 호버된 이미지 클릭시 Daum book으로 이동합니다.


![북 애니메이션](https://github.com/pvvng/book_with_react/assets/112927193/f2b7dbf6-79be-4d50-8fed-88b8eb3147dd)

### 2. 검색창에 책 이름을 입력하여 원하는 책을 검색할 수 있습니다.


![북 검색](https://github.com/pvvng/book_with_react/assets/112927193/5eb1ca47-98a3-48d2-af6e-ee0f2f7443d2)

### 3. 하단 더 불러오기 버튼을 클릭하여 책 데이터를 추가로 불러올 수 있습니다. 이는 React-Query의 종속성에 의한 추가적인 데이터 패칭 기능입니다. 버튼 상태가 변경될 때마다, 새롭게 데이터를 불러오고 불러온 데이터를 HTML에 보여줍니다.


![북 더 불러오기](https://github.com/pvvng/book_with_react/assets/112927193/77d115b6-a51d-46e7-9993-8d4e97d74639)

### 4. 만약 더 불러올 책 데이터가 없거나, input 입력값이 비어 있을 때는 alert를 띄워 예외처리를 해줍니다.


![북 예외처리](https://github.com/pvvng/book_with_react/assets/112927193/31bd27c4-ba7d-49f8-a6ed-8641b529201e)

### 5. 화살표 버튼을 눌러 페이지 최상단으로 이동 가능합니다.


![북 오르락](https://github.com/pvvng/book_with_react/assets/112927193/c79c5568-f38a-48dc-bea9-e2ea2cebfc3b)
